Alur Pendaftaran Test

Untuk Memenuhi "Peraturan Rektor Nomor 33 Tahun 2019, Pedoman Pelaksanaan English Language Proficincy Test (ELPT) Bagi Mahasiswa Prog Diploma, Sarjana, Magister."
Keterangan :

Test akan terbagi dalam beberapa jadwal yaitu hari Selasa, Kamis dan Jumat dengan beberapa ketentuan seperti berikut:

Setiap hari terbagi 3 sesi (sesi 1-3) yaitu:

Sesi 1 (08.00-10.00), Sesi 2 (10.30-12.30), Sesi 3 (13.15-15.15)

Jika hasil test lebih >= 450 maka dapat mengambil sertifikat. Jika hasil test < 450 dapat mendaftar ulang tes sebanyak 3x (jika 3x belum mencapai >= 450 maka harus ikut kursus)

Pelaksanaan test dilakukan Minimal 12 orang dan Mkasimal 15 orang per sesi

untuk pendaftaran dan pembayran bisa klik link REGISTER dibawah ini

*catatan : tidak ada expired date untuk sertifikat
